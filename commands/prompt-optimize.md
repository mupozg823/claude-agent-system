# Prompt Optimize - 프롬프트 최적화

AI 프롬프트(CLAUDE.md, 스킬, 시스템 프롬프트 등)를 최적화하세요.

## 최적화 영역

### 토큰 효율
- 중복 제거
- 약어/축약 표현 활용
- 불필요한 예시 제거 (1개로 충분한 곳에 3개 있는 경우)
- 조건문 병합

### 명확성
- 모호한 지시 → 구체적 지시
- "좋은 코드를 작성해" → "함수는 20줄 이내, 단일 책임 원칙 준수"
- 부정문 → 긍정문 전환

### 구조
- 우선순위 순 정렬 (중요한 것 먼저)
- 계층적 구조 (헤더 → 세부사항)
- 테이블/리스트 활용

### 효과
- 원하는 출력 형식 명시
- 예외 케이스 처리 규칙
- 체이닝/연쇄 지시 최적화

## 출력
1. 현재 프롬프트 분석 (토큰 수, 문제점)
2. 최적화된 버전
3. 변경 전후 비교 (토큰 절약량)

$ARGUMENTS
