# Eureka - 창의적 문제해결

문제를 **5가지 사고 프레임워크**로 분석하여 창의적 솔루션을 도출하세요.

## 프레임워크

### 🔄 역전 사고 (Inversion)
"이 문제를 해결하지 않으려면?" → 반대에서 접근

### 🧩 분해 사고 (Decomposition)
문제를 최소 단위로 쪼개고, 각각에 대한 독립 솔루션 도출

### 🔗 유추 사고 (Analogy)
다른 분야에서 유사한 문제가 어떻게 해결되었는지 참고

### ⚡ 제약 제거 (Constraint Removal)
"만약 제약이 없다면?" → 이상적 솔루션 후 현실에 맞게 조정

### 🎯 첫 원리 (First Principles)
근본 원리에서 출발하여 처음부터 재구성

## 출력
1. 각 프레임워크별 아이디어 (최소 1개씩)
2. 아이디어 교차 분석
3. **Top 3 실행 가능한 솔루션** (실행 난이도 + 효과 매트릭스)
