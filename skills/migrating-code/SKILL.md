# Code Migrate - 코드 마이그레이션

소스 프레임워크/라이브러리에서 타겟으로 코드를 마이그레이션하세요.

## 프로세스

### 1단계: 분석
- 현재 사용 중인 프레임워크/라이브러리 버전 확인
- 마이그레이션 대상 API/패턴 목록 작성
- 브레이킹 체인지 식별

### 2단계: 계획
- 변경 파일 목록 + 예상 변경 범위
- 마이그레이션 순서 (의존성 기반)
- 롤백 전략

### 3단계: 실행
- 파일별 변경 적용
- 타입 에러 해결
- 임포트 경로 업데이트

### 4단계: 검증
- 빌드 테스트
- 단위 테스트 실행
- 수동 확인 필요 항목 체크리스트

## 도구 활용
- context7으로 타겟 라이브러리 최신 문서 참조
- grep-app으로 마이그레이션 패턴 참고
- Serena로 심볼 기반 코드 수정
