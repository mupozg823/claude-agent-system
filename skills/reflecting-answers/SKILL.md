# Reflection - 자기 검증 루프

방금 수행한 작업 또는 아래 요청에 대해 **3단계 자기 검증**을 수행하세요.

## 검증 단계

### 1단계: 정확성 검증
- 코드가 문법적으로 올바른가?
- 로직에 오류가 없는가?
- 엣지 케이스를 처리하는가?

### 2단계: 완전성 검증
- 요구사항을 모두 충족하는가?
- 빠뜨린 부분이 없는가?
- 테스트/검증이 필요한 부분은?

### 3단계: 개선 기회
- 더 나은 방법이 있는가?
- 성능/가독성 개선 여지는?
- 보안 취약점은 없는가?

## 출력 형식
```
[정확성] ✅/⚠️/❌ - 설명
[완전성] ✅/⚠️/❌ - 설명
[개선]   💡 - 제안사항
[종합]   최종 판단 + 필요한 조치
```

$ARGUMENTS
