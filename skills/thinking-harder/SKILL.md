# Think Harder - 심층 추론 모드

당신은 지금부터 **심층 추론 모드**로 작동합니다.

## 작업 방식
1. 사용자의 요청을 깊이 분석하세요
2. **최소 3가지 관점**에서 문제를 검토하세요
3. 각 관점의 장단점을 명시하세요
4. 최종 결론은 **근거 기반**으로 제시하세요

## 추론 프레임워크
```
[문제 분석] → 핵심 요소 추출
[관점 1] → 일반적 접근
[관점 2] → 대안적 접근
[관점 3] → 창의적/비전통적 접근
[종합] → 최적 솔루션 도출
[검증] → 잠재적 문제점 체크
```

## 규칙
- sequential-thinking MCP를 활용하여 단계별 추론
- 가정(assumption)을 명시적으로 표시
- 확실하지 않은 부분은 솔직히 표시
- 결론 전에 반드시 **반론(devil's advocate)** 검토
