# Context Save - 작업 컨텍스트 저장

현재 세션의 작업 컨텍스트를 `~/.claude/contexts/`에 저장하세요.

## 저장할 내용
1. **현재 작업 요약**: 무엇을 하고 있었는지
2. **수정된 파일 목록**: 경로 + 변경 내용 요약
3. **발견한 이슈**: 아직 해결하지 않은 문제
4. **다음 단계**: 이어서 해야 할 작업
5. **핵심 결정사항**: 왜 이 접근을 선택했는지
6. **관련 파일 경로**: 주요 작업 대상 파일들

## 저장 형식
파일: `~/.claude/contexts/context-{YYYY-MM-DD}_{주제}.md`

```markdown
# Context: {주제}
- 저장 시각: {ISO timestamp}
- 작업 디렉토리: {cwd}

## 작업 요약
...

## 수정된 파일
...

## 미해결 이슈
...

## 다음 단계
...

## 핵심 결정
...
```

## 실행
1. `mkdir -p ~/.claude/contexts/` 디렉토리 생성
2. 현재 세션 대화 내용을 분석
3. 위 형식으로 저장
4. 저장 완료 메시지 출력

$ARGUMENTS
