#### 추가된 요소
- 시간대별 하늘 (낮/저녁노을/밤 + 별/달/구름)
- 건물 외장: 측벽 깊이감, 난간, 급수탑, AC 유닛(작업중 팬 회전)
- 네온사인 (전구 깜빡임 + 발광 효과)
- 안테나 + 연결 시 신호파 애니메이션
- 나무 4그루 + 잔디 + 포장도로 + 도로 차선 + 가로등(야간 발광)
- 정문 입구 (차양, 양문, 문손잡이, 계단)
- 층별 창문 (야간 실내 조명 반영)
- 층별 천장 조명 + 작업시 조명 확산
- 층별 고유 장식 (1F 서버랙 LED, 2F 책장+화이트보드, 3F 세계지도+실시간 시계)
- 미니 에이전트 치비: 눈깜빡임, 팔 애니메이션, 말풍선, 타원 그림자
- 엘리베이터: 케이블+레일+캐비닛 + 패킷 글로우/트레일
- 새 날아다니는 앰비언트 애니메이션
- 현재층 금색 화살표 표시
